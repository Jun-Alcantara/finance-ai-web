# Finance AI

Manage your finances with AI. This application provides a modern interface for tracking and analyzing your financial data.

## Features

- **User Authentication**: Secure login and registration functionality.
- **Financial Dashboard**: View and manage your financial status.
- **AI Integration**: AI-powered insights for BETTER financial decision making.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **HTTP Client**: [Axios](https://axios-http.com/)

## Getting Started

### Prerequisites

- Node.js installed on your machine.
- A running backend server at `http://localhost:8000` (API endpoint).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Application routes and pages.
- `lib/`: Utility libraries and API configuration.
- `public/`: Static assets.
